# Felix Portfolio

A lightweight personal portfolio built with Next.js App Router and Tailwind CSS.

## Overview

This project presents Felix Nshimyumukiza as a Full Stack Developer and Founder & CEO, with a strong focus on UI/UX, backend system thinking, and lightweight modern web development.

The portfolio includes:
- Hero section
- About Me
- Skills with visual progress bars
- Projects section
- Certifications
- Contact section
- Dark mode toggle
- Responsive layout
- Local assets only

## Deploy To Vercel

1. Push this repo to GitHub.
2. Import the repository into Vercel.
3. Keep the default framework as `Next.js`.
4. Vercel will use:

```text
Install Command: npm install
Build Command: npm run build
Output: managed automatically by Next.js
```

The project is configured for Node `20.x`, which matches Vercel's supported runtime for this app.
